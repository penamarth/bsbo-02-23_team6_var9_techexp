```
@startuml
actor "Инвестор" as Investor
actor "Эксперт" as Expert
participant "UI" as UI
participant "GrantStatisticsService" as StatsService
participant "StatisticsRepository" as StatsRepo
participant "Database" as DB

== Получение общей статистики ==

Investor -> UI: Запрос статистики
UI -> StatsService: getStatistics(filters)
StatsService -> StatsRepo: getGeneralStats(filters)
StatsRepo -> DB: запрос данных
DB --> StatsRepo: данные
StatsRepo --> StatsService: статистика
StatsService --> UI: результат
UI --> Investor: отображение статистики

== Статистика эксперта ==

Expert -> UI: Моя статистика
UI -> StatsService: getExpertStats(expertId)
StatsService -> StatsRepo: getExpertStatistics(expertId)
StatsRepo -> DB: запрос данных эксперта
DB --> StatsRepo: данные
StatsRepo --> StatsService: статистика эксперта
StatsService --> UI: результат
UI --> Expert: отображение статистики

== Генерация отчета ==

Investor -> UI: Создать отчет
UI -> StatsService: generateReport(filters, format)
StatsService -> StatsRepo: createReport(filters, format)
StatsRepo -> DB: сбор данных для отчета
DB --> StatsRepo: данные
StatsRepo --> StatsService: готовый отчет
StatsService --> UI: файл отчета
UI --> Investor: скачать отчет

@enduml
```
