### Рамки. Модуль "Статистика грантов" предоставляет аналитику по грантовым заявкам для анализа эффективности работы системы и принятия управленческих решений.
Уровень. Системный модуль для сбора, обработки и представления статистических данных.

### Основные исполнители:
- Инвестор - запрашивает общую статистику и отчеты
- Эксперт - просматривает персональную статистику работы

### Заинтересованные лица и их требования:
- Инвестор: Получение сводной статистики по грантам, анализ эффективности распределения средств, формирование отчетов
- Эксперт: Мониторинг личных показателей работы, анализ загрузки и эффективности

### Предусловия
- Пользователь аутентифицирован в системе 
- Имеются данные о грантовых заявках для анализа
- Пользователь имеет права доступа к статистике

### Результаты (Постусловия)
- Сформирована и отображена статистическая информация
- Сгенерированы отчеты в запрошенном формате
- Обеспечен доступ к данным согласно ролевой модели

### Основной успешный сценарий
- Запрос общей статистики:
  - Инвестор через UI запрашивает статистику с применением фильтров
  - Система обрабатывает запрос и возвращает агрегированные данные
  - Данные отображаются в удобном формате
- Получение экспертной статистики:
  - Эксперт запрашивает персональную статистику
  - Система предоставляет данные по рассмотренным заявкам и показателям работы
- Генерация отчетов:
  - Инвестор формирует отчет с применением фильтров
  - Система создает отчет в выбранном формате
  - Отчет становится доступен для скачивания

### Расширения (альтернативные потоки)
- Недостаточно данных для статистики - система возвращает сообщение об отсутствии данных
- Ошибка генерации отчета - система уведомляет о невозможности создания отчета

### Специальные требования
- Простой и интуитивный интерфейс для просмотра статистики
- Поддержка экспорта в различные форматы (PDF, Excel, CSV)
- Фильтрация данных по периодам и параметрам
- Быстрое выполнение запросов к статистике

### Список технологий и типов данных
- Агрегированные статистические данные
- Параметры фильтрации и форматы отчетов
- Данные о грантовых заявках и экспертах

### Частота использования
- Регулярное использование инвесторами для мониторинга
- Периодическое использование экспертами для самоанализа
- Формирование отчетов в периоды подведения итогов
