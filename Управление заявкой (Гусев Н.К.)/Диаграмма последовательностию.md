```
@startuml
actor "Соискатель гранта" as Applicant
participant "UI" as UI
participant "GrantApplicationFacade" as Facade
participant "IApplicantRepository" as ApplicantRepo
participant "IGrantApplicationRepository" as ApplicationRepo
participant "NotificationService" as NotifyService
participant "AuthorizationService" as AuthService
participant "ExpertiseAdapter" as ExpertiseAdapter
participant "ExpertiseServiceAPI" as ExpertiseAPI
participant "Database" as DB

Applicant -> UI: Создать заявку на грант
UI -> Facade: CreateApplication(applicantId: uInt32)
Facade -> AuthService: ValidateAccess(applicantId: uInt32)
AuthService --> Facade: AccessGranted

Facade -> ApplicantRepo: FindById(applicantId: uInt32)
ApplicantRepo -> DB: SELECT * FROM applicants WHERE id = applicantId
DB --> ApplicantRepo: Applicant data
ApplicantRepo --> Facade: Applicant object

Facade -> ApplicationRepo: Create(application: Application)
ApplicationRepo -> DB: INSERT INTO grant_applications (values)
DB --> ApplicationRepo : Application data
ApplicationRepo --> Facade: Application object

Facade -> UI: Application info
UI --> Applicant: Отображает статус и номер заявки

Applicant -> UI: Загружает документы к заявке
UI -> Facade: AttachDocuments(applicationId: uInt32, docs: File[])
Facade -> ApplicationRepo: AddDocuments(applicationId: uInt32, docs)
ApplicationRepo -> DB: INSERT INTO documents (values)
DB --> ApplicationRepo: Documents data
ApplicationRepo --> Facade: Документы прикреплены

Facade -> ExpertiseAdapter: SubmitForExpertise(applicationId: uInt32)
ExpertiseAdapter -> ExpertiseAPI: submitApplication(applicationId)
ExpertiseAPI --> ExpertiseAdapter: expertiseResult
ExpertiseAdapter --> Facade: expertiseResult

Facade -> NotifyService: Notify(applicantId, "Заявка принята и направлена на экспертизу")
NotifyService --> Facade: Confirmation

Facade -> UI: Показывает обновленный статус заявки
UI --> Applicant: Статус: "На экспертизе"
@enduml
```
